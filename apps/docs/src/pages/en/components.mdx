## Adding click-to-zoom to Images

Be sure you properly import the image you are wanting to wrap. This is only to 
be used for screenshots or diagrams (nothing small like icons). It also 
requires you to add the following statements at the top of the mdx file:

```javascript
import RichImg from "@components/Misc/RichImg.astro";
```

Then, call the component like:

```javascript
<RichImg src={yourImage} alt="yourImage information" />
```
